# Day 04 - File Permission and Ownership

## Objective
- To understand what file permision is
- To know how to change file permissions
- Understand types of users



## What I Learned

- I learnt that there are three type of people who can access a file (user, group, others)
- the file permision contains 10 characters (1 - file type [- for file, d   for directory and l for link)
- characters 2 - 4 (users permision), characters 5-7(group permisions) and 8-10 (others permission)
- permisions can either be read (r), write(w), execute (x)
- the command for changing permission is chmod (change mode0
- the command for changing permission is chown (change ownership)
